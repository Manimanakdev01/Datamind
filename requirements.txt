#  NOTES FOR STREAMLIT CLOUD

#  1. Python version — add a .python-version file in your repo root:
#       echo "3.11" > .python-version

#  2. TensorFlow on Streamlit Cloud uses ~1 GB RAM.
#     If you hit memory limits, replace the tensorflow lines above with:
#       tensorflow-cpu>=2.15.0,<2.17.0

#  3. Ultralytics pulls in torch (~700 MB). If you don't use YOLO,
#     comment out the ultralytics line to keep the slug small.
#
#  4. Firebase secrets — store in Streamlit Cloud > App settings > Secrets:
#       [firebase]
#       api_key      = "..."
#       database_url = "..."
#       ...
#     Then read with st.secrets["firebase"]["api_key"] and update _FB_CONFIG.
#
#  5. Razorpay secrets — same approach:
#       [razorpay]
#       key_id     = "rzp_live_..."
#       key_secret = "..."
#     Then at the top of main.py:
#       RAZORPAY_KEY_ID     = st.secrets["razorpay"]["key_id"]
#       RAZORPAY_KEY_SECRET = st.secrets["razorpay"]["key_secret"]
